# script-lol

🌙 Dark Mode Universal

Ativa o modo escuro em qualquer site com apenas um clique. Inverte cores de forma inteligente, protegendo seus olhos durante a noite. Funciona em YouTube, Google e qualquer página clara.

📋 Copy Unlocker

Desbloqueia sites que impedem cópia de texto. Remove todas as restrições de seleção, clique direito e copiar/colar. Ideal para pesquisas e trabalhos acadêmicos.

🔗 Link Collector

Coleta e organiza todos os links da página atual em ordem alfabética. Gera um arquivo .txt organizado com numeração. Perfeito para curadoria de conteúdo e pesquisas.

🚀 Brutal Site Opener

Abre qualquer site bloqueado usando múltiplas tentativas: iframe, nova janela e redirect. O modo mais agressivo para furar bloqueios de rede.

🦁 Brave Installer

Baixa automaticamente o instalador do Brave Browser. Abre o site oficial e inicia o download em segundos. Para quem quer um navegador mais rápido e seguro.

# Todas as ferramentas desenvolvidas por vicav_vlr com sistema de aviso visual de 4 segundos e indicação de sucesso/erro na execução.

# MODO DE USO

No Chrome/Edge/Brave/Vivaldi:

1. Clique na estrela ⭐ na barra de endereços
2. Clique em "Mais..." ou "Editar"
3. No campo "Nome" : coloque o nome da ferramenta (ex: "🌙 Dark Mode")
4. No campo "URL" : cole TODO o código do script
5. Clique em "Salvar"

No Firefox:

1. Clique na estrela ⭐ na barra de endereços
2. Clique em "Editar este favorito"
3. No campo "Nome" : coloque o nome
4. No campo "Endereço" : cole TODO o código
5. Clique em "Salvar"
